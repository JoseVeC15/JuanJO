# FINANCE PRO Web

Esta es la versión web de FINANCE PRO.

## Guía de uso

1. Ingresa a la plataforma desde tu navegador.
2. Inicia sesión con tus credenciales.
3. Desde el panel principal puedes:
   - Consultar y gestionar facturas.
   - Administrar el inventario.
   - Revisar reportes y estadísticas.
   - Gestionar proyectos.
4. Utiliza el menú lateral para navegar entre las secciones.
5. Para cerrar sesión, haz clic en tu perfil y selecciona "Cerrar sesión".

Si tienes dudas, consulta con el administrador o revisa la ayuda dentro de la plataforma.
