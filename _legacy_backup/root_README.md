# FINANCE PRO

Bienvenido a FINANCE PRO, la plataforma integral para la gestión de facturas, inventario, reportes y proyectos.

## Opciones disponibles

- **FINANCE PRO Web**: Accede desde tu navegador para gestionar tus operaciones de manera sencilla.
- **FINANCE PRO Alternativa**: Una versión alternativa con diferente presentación, pero las mismas funciones principales.
- **FINANCE PRO Audiovisual**: Solución para proyectos y recursos audiovisuales, con acceso web y móvil.

Para comenzar, ingresa a la carpeta de la opción que prefieras y sigue la guía de uso incluida en cada una.
