# FINANCE PRO Audiovisual

FINANCE PRO Audiovisual está diseñado para la gestión de proyectos y recursos audiovisuales, con acceso web y móvil.

## Guía de uso

1. Ingresa desde la web o la app móvil.
2. Inicia sesión con tu cuenta.
3. Gestiona proyectos, inventario y facturación desde el menú principal.
4. Utiliza las funciones de cámara y confirmación en la app móvil para registrar recursos.
5. Consulta reportes y administra usuarios según tus permisos.

Si necesitas ayuda, revisa la sección de ayuda o contacta al administrador.
