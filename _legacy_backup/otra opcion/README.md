# FINANCE PRO Alternativa

Esta es una versión alternativa de FINANCE PRO, con las mismas funciones principales y una presentación diferente.

## Guía de uso

1. Abre la plataforma desde tu navegador.
2. Inicia sesión con tu usuario y contraseña.
3. Accede a las secciones de facturas, inventario, reportes y proyectos desde el menú principal.
4. Navega entre las opciones para gestionar tu información.
5. Cierra sesión desde el menú de usuario cuando termines.

Para soporte, contacta al administrador o revisa la ayuda en la plataforma.
